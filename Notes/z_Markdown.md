### 代码与代码块
```c
#include<stdio.h>
#include<math.h>

int main()
{
    int n,i,k,r;
    while(1){
        scanf("%d",&n);
    if (n==2)
        printf("2 is a prime\n");
    else
    {
        i=1;
        k=sqrt(n);
        do{
            ++i; r=n%i;
        }while(r&&i<=k);
        if(r)
          printf("%d is a prime.\n",n);
        else
          printf("%d isn't a prime.\n",n);
    }
    }
    return 0;
}
```
要指定语言类型才能显示高亮  
在行内输入代码:`for(i = 0;i < n;++i)`
